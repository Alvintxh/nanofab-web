# Architecture

## Overview

NanoFab Learning Platform 是一个基于《纳米制造技术：原理、工艺与实践》的AI驱动个性化学习网站。系统提供静态标准版和动态个人定制版两种学习模式。

## Key Components

### 1. 用户画像系统 (User Profile)
- **输入**: 学习动机、知识背景、上传材料（简历、成绩单等）
- **处理**: AI分析生成用户画像
- **输出**: 个性化学习路径和内容定制参数

### 2. 内容呈现系统 (Content Delivery)
- **静态版本**: 完整的12章内容，结构化的知识库
- **动态版本**: 基于用户画像的定制化内容（增删改）
- **内容格式**: Markdown → HTML，支持数学公式、图表、代码块

### 3. AI解释系统 (AI Explanation)
- **触发**: 用户高亮文本并请求解释
- **上下文**: 用户画像 + 当前章节内容 + 高亮文本
- **输出**: 定制化的深度解释

### 4. 行为追踪系统 (Behavior Tracking)
- **收集**: 页面停留时间、滚动行为、点击热图、Quiz表现
- **分析**: 识别用户关注点和薄弱环节
- **反馈**: 动态调整用户画像和学习路径

### 5. 学习辅助组件 (Learning Aids)
- **要点卡片**: 每章核心知识点提炼
- **Quiz**: 递进式问题引导深入思考
- **思考练习**: 巩固和回顾

## Data Flow

```
用户注册 → 收集画像信息 → AI生成画像 → 定制学习路径
                ↓
        学习行为数据 ← 实时追踪 ← 用户交互
                ↓
        动态调整画像 → 更新内容呈现
```

## Key Decisions

| Decision | Rationale | Date |
|----------|-----------|------|
| 纯前端架构（HTML/CSS/JS） | 快速原型，无需后端部署 | 2026-05-04 |
| LocalStorage存储用户数据 | 简化初期实现，后续可迁移 | 2026-05-04 |
| 从现有Markdown知识书提取内容 | 复用已有内容，减少重复工作 | 2026-05-04 |
