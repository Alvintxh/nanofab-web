# 使用手册

## 概述

纳米制造技术学习平台是基于《纳米制造技术：原理、工艺与实践》的 AI 驱动个性化学习系统。本手册覆盖所有功能的使用方式。

## 快速开始

1. 打开网站，浏览欢迎页了解平台特色
2. 点击"开始学习"，注册账号（邮箱+密码）
3. 填写用户画像（学习水平、专业背景、学习动机等）
4. 进入首页仪表盘，开始学习

## 功能说明

### 课程学习

| 功能 | 如何使用 | 位置 |
|------|---------|------|
| 章节阅读 | 左侧导航栏选择章节，中间区域显示内容 | 全局 |
| 章首速览 | 每章顶部自动显示 TL;DR 摘要 | 章节页顶部 |
| 学习目标 | 每章附带基础/进阶学习目标 | TL;DR 下方 |
| 章节导航 | 底部"上一章/下一章"按钮，或侧边栏直接跳转 | 章节底部 |
| 完成标记 | 阅读完点击底部"标记完成"按钮 | 章节底部 |
| 图片放大 | 点击章节内任意图片 | 章节内容区 |

### 学习工具

| 功能 | 如何使用 | 位置 |
|------|---------|------|
| 要点卡片 | 章节内自动展示核心知识点 | 章节 TL;DR 下方 |
| 交互式测验 | 每章 10 题（单选/多选/判断），即时反馈 | 章节底部"测验"标签 |
| 思考练习 | 开放式问题，点击展开提示 | 章节底部"思考"标签 |
| 思维导图 | 首页"课程结构"卡片，展示 4 部分 12 章树状结构 | 首页仪表盘 |

### AI 功能

#### AI 文本解释

1. 在章节内容中**选中任意文本**
2. 弹出浮动按钮，点击"AI 解释"
3. 在输入框中输入你的问题（可选，留空则 AI 自动解释）
4. AI 根据你的知识水平给出个性化解释
5. 可在结果下方**继续追问**，保持上下文
6. 点击"保存全部对话为笔记"将 Q&A 保存

#### AI 学习助手

1. 点击右上角"AI 助手"按钮打开侧边栏
2. 直接输入任何学习相关问题
3. AI 会自动结合你的笔记、错题、学习进度等数据回答
4. 支持 Markdown 格式回复（标题、列表、粗体等）

#### AI 学习路线

- 首页"AI 学习路线"卡片自动基于你的画像和进度生成推荐路径
- 点击刷新可重新生成

### 笔记系统

| 操作 | 方法 |
|------|------|
| 添加笔记 | 选中文本 → 点击高亮笔图标 → 输入笔记内容 → 保存 |
| 查看笔记 | 点击已高亮的文本 → 显示内联注解 |
| AI 笔记 | AI 解释 → 点击"保存全部对话为笔记" → 原文自动高亮 |
| 管理笔记 | 右上角用户头像 → "我的笔记" → 查看/删除 |

### 错题本

- 首页"错题本"卡片自动收集所有答错的测验题
- 点击可复习错题及正确答案

### 学习统计

- 首页"学习统计"卡片展示：
  - 今日/本周/总计学习时长
  - 答题正确率
  - 连续学习天数
  - 周目标进度环

### 用户设置

| 操作 | 方法 |
|------|------|
| 查看/编辑资料 | 右上角用户头像 → 查看画像 |
| 切换 AI 提供商 | AI 助手侧边栏设置（智谱/DeepSeek/Gemini） |
| 重置学习数据 | 用户资料弹窗底部"重置数据" |
| 退出登录 | 用户资料弹窗底部"退出登录" |

## 常见问题

**Q: 忘记密码怎么办？**
登录表单下方点击"忘记密码"，输入邮箱，查收重置邮件。

**Q: AI 解释不显示怎么办？**
检查 AI 提供商设置是否正确。智谱 AI 使用服务端 Key，DeepSeek/Gemini 需要自行填入 API Key。

**Q: 笔记保存在哪里？**
笔记保存在浏览器本地存储和 Supabase 数据库（登录后自动同步）。

**Q: 如何查看学习进度？**
首页"学习进度"卡片显示完成百分比和进度环，侧边栏章节前有完成标记。
