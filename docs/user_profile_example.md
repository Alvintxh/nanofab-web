# 用户画像示例

## 示例用户：XH

### 1. 学习动机

**主要目标**: 设计和制造X射线衍射光学元件（X-ray Diffractive Optical Elements）

**具体应用场景**:
- 同步辐射光束线中的聚焦和成像光学元件
- 高分辨率X射线显微镜的物镜
- 需要亚微米级精度的衍射光栅和波带片

**期望成果**:
- 掌握电子束光刻（EBL）在X射线光学元件制造中的应用
- 理解纳米级图形转移工艺（刻蚀、沉积、lift-off）
- 能够独立设计和优化工艺配方

### 2. 知识背景

**教育背景**:
- 博士：光学工程 / 同步辐射应用
- 硕士：物理学（凝聚态物理方向）
- 本科：应用物理学

**专业技能**:
- **光学**: 精通X射线光学、衍射理论、傅里叶光学
- **物理**: 固体物理、量子力学、电磁学基础扎实
- **编程**: Python（数据分析、科学计算）、MATLAB
- **实验**: 同步辐射实验经验、光学系统调试

**薄弱环节**:
- **材料科学**: 对光刻胶化学、薄膜材料性质了解有限
- **半导体工艺**: 缺乏IC制造产线经验，对工艺集成理解不深
- **设备操作**: 未操作过电子束光刻机、刻蚀机等纳米加工设备

**相关经验**:
- 使用过光学光刻制作宏观尺度的衍射元件（特征尺寸 > 1 μm）
- 了解基本的薄膜沉积概念（热蒸发、溅射），但未实际操作
- 阅读过纳米制造相关文献，但缺乏系统性知识

### 3. 学习偏好

**学习风格**:
- 偏好从物理原理出发理解工艺
- 需要看到数学公式和定量分析
- 喜欢通过对比不同技术的优缺点来加深理解

**时间投入**:
- 每周可投入 10-15 小时
- 希望 3-6 个月内掌握核心工艺

**内容偏好**:
- 重点关注与X射线光学元件直接相关的章节
- 需要更多关于EBL、lift-off工艺、金属沉积的细节
- 对IC制造中的多层互连等内容兴趣较低

## 画像生成逻辑

### 定制内容策略

**对于强物理背景用户**:
- 光学光刻章节：重点解释衍射极限的物理本质，而非工艺参数
- EBL章节：深入电子散射物理、邻近效应的数学模型
- 刻蚀章节：强调等离子体物理和表面反应动力学

**对于弱材料背景用户**:
- 增加光刻胶化学基础介绍
- 补充薄膜材料选择的基本原则
- 提供更多材料兼容性表格

**对于X射线光学目标**:
- 在相关章节增加X射线光学元件的案例
- 强调高深宽比结构、金属lift-off等关键工艺
- 关联同步辐射应用的实际需求
