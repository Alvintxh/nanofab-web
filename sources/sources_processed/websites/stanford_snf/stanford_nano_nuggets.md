# Stanford Nanofabrication Facility - Nano Nuggets Guide

## Overview

The Stanford Nanofabrication Facility (SNF) maintains a collection of practical guides called "Nano Nuggets" designed to help researchers with various fabrication techniques. These represent "useful bits of information about different processing methods."

## Available Nano Nuggets Topics

### Polymer Processing
- PDMS (polydimethylsiloxane) wafer protection techniques using blue sticky tape
- Drop gauge measurement procedures for PDMS film thickness
- Mold creation protocols for thick, uniform PDMS casting
- Hydrogel bilayer fabrication with pNIPAm-AAc and SU-8 photoresists

### Equipment-Specific Procedures
- Aixtron BM cleaning protocols
- Nanoscribe 3D printing troubleshooting and optimization
- Microlens printing procedures
- PT-DSE (plasma-based) structure release processes

### Bonding and Adhesion
- Plasma-activated direct bonding techniques using the Flipchip tool
- Manual tweezers-based bonding for silicon and lithium niobate
- Gold-PDMS adhesion enhancement strategies

### Nanofabrication Techniques
- Sub-micron metal patterning on polymers via nanostencil lithography
- Cuprite growth procedures
- Etch rate characterization methods

### Characterization Methods
- Drop gauge measurements for etch rate analysis
- SEM-based analysis of partially released structures

## Navigation and Access

The Nano Nuggets are paginated, with multiple pages of resources available. Users can search by title and filter by equipment type. Each nugget typically includes step-by-step procedures, practical tips, and links to detailed project reports when applicable.

The facility emphasizes hands-on learning, with video demonstrations from E241 student projects included alongside standard operating procedures.
