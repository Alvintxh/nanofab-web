# Stanford Nanofabrication Facility - Processes

## Overview

The Processes section of Stanford Nanofabrication Facility's Lab User Guide provides documentation for common lab procedures.

## Processes Listed

The current Processes section includes six primary processes:

1. **Dry Etching**: Carrier wafer attachment procedures
2. **Dry Resist Stripping**: Descum and polymer removal techniques
3. **Photoresist Stripping**: Options based on cleanliness requirements
4. **Tweezers Cleaning**: Equipment maintenance
5. **Wet Cleaning (Pre-Diffusion)**: SC1/HF/SC2 protocol for oxidation
6. **Wet Cleaning (Pre-LPCVD)**: Standard cleaning before deposition

## Key Resources

The guide integrates with:
- NEMO (lab scheduling system requiring login)
- Equipment categorization by technique (CVD, doping, dry etch, metallization, photolithography, wet chemical processing)
- Material safety data sheets and chemical inventories
- Device process course documentation (EE312/EE410)
