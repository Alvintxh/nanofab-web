# SiNx PECVD and Nanostructure Etching Recipe Development

## Project Overview

This project, completed in June 2014, focused on developing recipes for silicon nitride deposition and nanostructure etching processes at Stanford Nanofabrication Facility.

## Key Information

**Project Type:** E241 (Fab Project Course)

**Research Focus:** "PECVD SiNx stress, Nanosphere etching"

**Researchers:** Yusi Chen and Muyu Xue

## Equipment Used

The researchers employed two primary tools:

1. **PlasmaTherm Shuttlelock PECVD System (ccp-dep)** -- for silicon nitride deposition
2. **Plasma Therm Versaline LL ICP Dielectric Etcher (PT-Ox)** -- for dry etching processes

## Processing Techniques

The project integrated two complementary fabrication methods:

- Plasma Enhanced (PE) CVD
- Dry Etching

## Documentation

The project generated comprehensive deliverables:

- **Final Report:** SiNx PECVD and Nanostructure Etching Recipe Development - Final Report
- **Final Presentation:** SiNx Deposition and Etching Recipe Development - Final Presentation

## Context

This work falls within the Stanford Nanofabrication Facility's broader offerings in chemical vapor deposition and dry etching capabilities, supporting both educational and research objectives in nanostructure fabrication.
