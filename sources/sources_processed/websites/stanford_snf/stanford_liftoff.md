# Metal Lift-off - Stanford Nanofabrication Facility

## Overview

Metal lift-off is a patterning technique that uses a sacrificial layer to define metallic structures. The process involves several sequential steps:

1. **Resist Application & Patterning**: A sacrificial layer (typically photoresist) is applied and optically patterned
2. **Material Deposition**: The target material (commonly a metal) is deposited across the entire surface
3. **Lift-off**: The sacrificial layer is dissolved, removing any deposited metal that sits atop it, while leaving the patterned metal on the substrate
