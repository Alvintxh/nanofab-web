# Silicon Nitride - Stanford Nanofabrication Facility

## Overview

Silicon Nitride (Si3N4) is a commonly used material in semiconductor fabrication. It functions as "an insulator in traditional semiconductors, a masking layer or a structural layer in some mechanical applications."

## Deposition Methods

The facility offers multiple techniques for depositing silicon nitride:

- **LPCVD**: Thermco Nitride Deposition Furnace
- **PECVD**: PlasmaTherm Shuttlelock system using silane and nitrogen reactions
- **HDPCVD**: PlasmaTherm Versaline system with 100% silane and N2
- **ALD**: Atomic layer deposition via Fiji 2 ALD

## Etching Equipment

Silicon nitride can be etched using:

- MRC Reactive Ion Etcher
- Oxford Dielectric Etcher
- Oxford Plasma Pro ICP-RIE systems
- Plasma Therm Versaline LL ICP Dielectric Etcher
- Samco PC300 Plasma Etch System
- Wet Bench Clean_res-hotphos
- Wet Bench Flexcorr 1

A dedicated [Silicon Nitride Wet Etching Technique guide](https://snfguide.stanford.edu/guide/equipment/purpose/etching/wet-etching/silicon-nitride-wet-etching) is available.
