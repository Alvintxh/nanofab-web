# Dry Etching Recipes - UCSB Nanofab Wiki

## Overview

This wiki page catalogs dry etching processes and materials available at UCSB's nanofabrication facility.

## Process Control Data
References statistical process control data (deposition rates, stress measurements over time) for frequently-used dry etches on a linked page.

## Dry Etching Tools/Materials Table

The page features a comprehensive matrix showing available etching equipment and compatible materials:

**Equipment Categories:**
- RIE (Reactive Ion Etching): RIE 2 (MRC), RIE 5 (PlasmaTherm)
- ICP (Inductively Coupled Plasma): DSEIII, Fluorine ICP, Panasonic models, Oxford ICP
- Oxygen Plasma Systems: Ashers, Plasma Clean, UV Ozone Reactor, Plasma Activation
- Other Systems: XeF2 Etch, Vapor HF Etch, CAIBE

**Materials Covered:**
Metals (Al, Au, Cr, Cu, Mo, Ni, Pt, Ru, Si, Ta, Ti, W, TiW), semiconductors (GaAs, GaN, InP), oxides (SiO2, Al2O3), nitrides (SiN, AlN, TiN), and organic materials.

## Process Maturity Ranking

Processes are ranked from R6 (most mature with regular calibration records) to R1 (run only once), plus "A" for allowed but never performed.

### Process Ranking Levels

| Level | Description |
|-------|-------------|
| A | Allowed; materials available but never executed |
| R1 | Executed at least once |
| R2 | Executed with documented procedure |
| R3 | Executed, documented, with available data |
| R4 | Documented with regular data (>=4x yearly); no in-situ control |
| R5 | Documented with regular data and in-situ control available |
| R6 | Documented procedure and control charts/limits available |
